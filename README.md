# SelenicChroniclesTicTacToe

找出一种井字棋变体的所有状态。该变体出现在《原神》的「伴月纪闻」中。在此变体中，玩家三回合前的棋子会在玩家准备落子之前消失。

Find all states of a tic-tac-toe variant. This variant appears in the Selenic Chronicles of Genshin Impact. In this variation, a player's place from three turns ago disappears before the player prepares to place.
